AI-Chatbot

AI-Chatbot es una aplicación sencilla desarrollada en JavaScript que funciona como un asistente automatizado para responder preguntas frecuentes.



El chatbot analiza la entrada del usuario y devuelve respuestas basadas en coincidencias con una base de datos de preguntas predefinidas. Está diseñado para ofrecer ayuda inmediata, mejorar la atención al usuario y reducir la carga de soporte. Su estructura permite agregar nuevas preguntas y respuestas fácilmente, haciéndolo ideal para sitios que necesitan una solución rápida y ligera sin depender de APIs externas o modelos avanzados de IA.
