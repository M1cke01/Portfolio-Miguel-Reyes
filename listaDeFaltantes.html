Lista de Faltantes

Lista de Faltantes es una aplicación diseñada para coordinar y optimizar las necesidades de inventario entre varias sucursales.

Cada tienda puede registrar los productos que le hacen falta y visualizar qué otras sucursales cuentan con ese material. A través de un selector intuitivo, las tiendas pueden indicar si tienen el producto disponible, si ya lo enviaron o si está en ruta. Además, el repartidor puede actualizar el estatus o marcar como resuelto cada faltante. El sistema permite que todas las sucursales vean las listas de todas, mejorando la comunicación interna, evitando pedidos innecesarias y agilizando el abastecimiento en toda la red de tiendas.
