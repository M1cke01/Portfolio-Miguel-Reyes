PokeApi

PokeApi con TypeScript es una aplicación tipo Pokedex desarrollada en React utilizando TypeScript para garantizar mayor seguridad y control en los tipos de datos.


La app consume la PokeAPI para obtener información de cada Pokémon y permite buscarlos por nombre o ID. Muestra datos como imágenes, estadísticas, tipos y habilidades, todo en una interfaz dinámica y responsiva. El proyecto destaca por el uso de componentes funcionales, tipados estrictos con interfaces y tipos personalizados, manejo de estados con Hooks y llamadas a APIs externas. Es una excelente práctica de integración entre React, TypeScript y una API pública.
